## üí° Design Tips

### Using the Palette
1. **Backgrounds**: Use lightest grays (#f8f9fa, #e9ecef)
2. **Text**: Use darkest grays (#212529, #343a40)
3. **Borders**: Use middle grays (#dee2e6, #ced4da)
4. **Accents**: Always use teal (#4EC0D6)
5. **Hierarchy**: Vary gray darkness for importance

### Color Psychology
- **Grays**: Professional, neutral, sophisticated
- **Teal Accent**: Trustworthy, calming, modern
- **Combination**: Perfect for sports/booking apps
- **Appeal**: Wide audience, both genders, all ages
